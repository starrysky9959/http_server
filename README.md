# ucas_network_lab1

## 实验要求
实现：使用C语言实现最简单的HTTP服务器
同时支持HTTP（80端口）和HTTPS（443端口）
使用两个线程分别监听各自端口
只需支持GET方法，解析请求报文，返回相应应答及内容
